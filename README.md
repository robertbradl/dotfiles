## Requirements

- Some Nerd Font (default is GeistMono)
- nvim
- WezTerm
- Sketchybar
- Sketchybar App Font
- JankyBorders
- Yabai
- tmux
- SF Symbols
- SF Pro Font
- SF Mono Font

... for a full list of dependencies, check the
![install.sh](https://github.com/robertbradl/dotfiles/blob/main/install.sh) script.
